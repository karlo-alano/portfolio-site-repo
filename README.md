# My Portfolio

Created with Vue

## About this project

This project is supposed to be a practical application of the DevOps and System Administration related knowledge that I have. With multiple plans and multiple stages in the works, this project is 
poised to become a monolith of my technical skills. 

## Plans for the Future

The project is planned to be developed in multiple stages each with their own project deadlines and learning accomplishments.

1. Installation and initial setup (Completed) -- The initial stage where the operating system is set up as well as the installation of basic software utilities for efficiency and convenience.
2. Wireless connection (Completed) --  The stage where SSH is setup and intercomputer communication is established. Passwordless logins are established to promote better software security practices.
3. Hosting a static website (Completed) -- The stage where multiple services is interconnected to establish a foundation fit for hosting a static website. The `cloudflared` service is an essential service to
bypass CGNAT by tunneling blocked ingoing traffic outwards. Next, the NGINX service is used to establish a reversed proxy that serves the built website.
4. Implementing Github-hosted and Self-hosted runners for automation -- The stage where automation is taken into account in order to automatically build and serve the new website upon a `git push` to the repository.
5. Hosting a dynamic website -- The stage where the scope is expanded to host multiple services containerized using Docker and Docker Compose.

## About the Hardware

The server is run on a 15 year old Lenovo Thinkpad Edge e420. It currently has an 2nd-gen Intel Core i5-2450m, 4 gigabytes of DDR3 ram, and a 500-gigabyte 5200rpm Hard Disk.
